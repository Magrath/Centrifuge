Centrifuge by Magrath

Open source, multi-platform Web 2.0 Media Manager

Dependencies

  • python2
  • python-hachoir-core
  • python-hachoir-metadata
  • python-hachoir-parser
  • python-eyed3
  • python-mysqldb
  • ffmpeg
  • python-imaging

Install

1) Pull copy from git repository.
2) Put www into an Apache2 vhost
3) Populate the database with mysql_population.txt
4) Install the packages listed in python-requirements.txt
5) Run spider.py
